# Ecossistema

O ecossistema do Nest é feita com diversas tecnologias já feitas e consolidadas no mercado para que aja de forma mais rápido e tranquilo do desenvolvimento da aplicação a ser criada

## Ferramentas terceiras

- **TypORM:** Para interagir com banco de dados
- **GraphQL:** Para construir APIs com GraphQL
- **Password:** Para autenticação
- **Swagger:** Para documentação da API

## Nest CLI

NestJS prove uma CLI poderosa para lidar com os processos de desenvolvimento

### Beneficios 

- Fácil de gerar módulos, services, e controllers assim temos uma maneira mais tranquila de sermos preguiçosos
- Build e serve: compile e roda sua aplicação com simples comandos

**Exemplo:**

```sh
nest new my-new-project
nest generate module users
````

> O primeiro comando serve para dar um setup básico no projeto e nomear com my-new-project `nest new my-new-project`x
> o Segundo vai gerar um model depois da criação do projeto, antes de rodar isso você precisaria estar dentro do diretorio do NestJS `nest generate module users`
